#### Usage

How to install, instantiate, or embed the footer objects.

##### Global Script

    <!-- Global Script -->
    <script src="dist/js/patterns.js"></script>

    <script>
      var Patterns = new Patterns();
      var footer = Patterns.footer();
    </script>

##### Module Import

    import Footer from 'src/objects/footer/footer';

    new Footer();
