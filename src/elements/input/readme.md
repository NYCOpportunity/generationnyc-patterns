#### Usage

How to install, instantiate, or embed the input elements.

##### Global Script

    <!-- Global Script -->
    <script src="dist/js/patterns.js"></script>

    <script>
      var Patterns = new Patterns();
      var input = Patterns.input();
    </script>

##### Module Import

    import Input from 'src/elements/input/input';

    new Input();
