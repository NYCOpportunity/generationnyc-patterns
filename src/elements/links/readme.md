#### Usage

How to install, instantiate, or embed the links elements.

##### Global Script

    <!-- Global Script -->
    <script src="dist/js/patterns.js"></script>

    <script>
      var Patterns = new Patterns();
      var links = Patterns.links();
    </script>

##### Module Import

    import Links from 'src/elements/links/links';

    new Links();
