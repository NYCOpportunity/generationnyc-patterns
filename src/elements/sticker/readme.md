#### Usage

How to install, instantiate, or embed the sticker elements.

##### Global Script

    <!-- Global Script -->
    <script src="dist/js/patterns.js"></script>

    <script>
      var Patterns = new Patterns();
      var sticker = Patterns.sticker();
    </script>

##### Module Import

    import Sticker from 'src/elements/sticker/sticker';

    new Sticker();
