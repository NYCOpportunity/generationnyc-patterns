#### Usage

How to install, instantiate, or embed the layout elements.

##### Global Script

    <!-- Global Script -->
    <script src="dist/js/patterns.js"></script>

    <script>
      var Patterns = new Patterns();
      var layout = Patterns.layout();
    </script>

##### Module Import

    import Layout from 'src/elements/layout/layout';

    new Layout();
