#### Usage

How to install, instantiate, or embed the list elements.

##### Global Script

    <!-- Global Script -->
    <script src="dist/js/patterns.js"></script>

    <script>
      var Patterns = new Patterns();
      var list = Patterns.list();
    </script>

##### Module Import

    import List from 'src/elements/list/list';

    new List();
