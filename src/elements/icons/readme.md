#### Usage

How to install, instantiate, or embed the icons elements.

##### Global Script

    <!-- Global Script -->
    <script src="dist/js/patterns.js"></script>

    <script>
      var Patterns = new Patterns();
      var icons = Patterns.icons();
    </script>

##### Module Import

    import Icons from 'src/elements/icons/icons';

    new Icons();
