#### Usage

How to install, instantiate, or embed the buttons elements.

##### Global Script

    <!-- Global Script -->
    <script src="dist/js/patterns.js"></script>

    <script>
      var Patterns = new Patterns();
      var buttons = Patterns.buttons();
    </script>

##### Module Import

    import Buttons from 'src/elements/buttons/buttons';

    new Buttons();
