#### Usage

How to install, instantiate, or embed the anchor utilities.

##### Global Script

    <!-- Global Script -->
    <script src="dist/js/patterns.js"></script>

    <script>
      var Patterns = new Patterns();
      var anchor = Patterns.anchor();
    </script>

##### Module Import

    import Anchor from 'src/utilities/anchor/anchor';

    new Anchor();
