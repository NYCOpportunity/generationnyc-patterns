#### Usage

How to install, instantiate, or embed the hero components.

##### Global Script

    <!-- Global Script -->
    <script src="dist/js/patterns.js"></script>

    <script>
      var Patterns = new Patterns();
      var hero = Patterns.hero();
    </script>

##### Module Import

    import Hero from 'src/components/hero/hero';

    new Hero();
