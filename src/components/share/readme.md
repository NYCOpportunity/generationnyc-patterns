#### Usage

How to install, instantiate, or embed the share components.

##### Global Script

    <!-- Global Script -->
    <script src="dist/js/patterns.js"></script>

    <script>
      var Patterns = new Patterns();
      var share = Patterns.share();
    </script>

##### Module Import

    import Share from 'src/components/share/share';

    new Share();
